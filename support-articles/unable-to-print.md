# Printer Issue

## Symptoms

* Printer is offline
* Unable to print documents
* Print jobs are stuck in queue
* Poor print quality or paper jams

## Resolution

1. Ensure the printer is powered on
2. Check cable or network connectivity
3. Verify the printer is set as the default printer
4. Clear pending print jobs from the queue
5. Restart the printer and system
6. Contact support if issue persists
